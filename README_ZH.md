- 瀏覽器

- HTTP 伺服器

- npm

## 基本建置

安裝 `http-server` 套件
```bash
$ npm install -g http-server
```

安裝用來編譯 sass 的 `sass` 套件
```bash
$ npm install -g sass
```

Clone 這個專案
```bash
$ git clone https://github.com/daancsc/DACSC.git
```

編譯 `style.sass` 成 `style.css`
```bash
$ cd DACSC
$ sass style.sass style.css
```

執行 http server
```bash
$ http-server
```
